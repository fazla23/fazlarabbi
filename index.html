Building ML based Firewall on STP over SDN

Project brief:

Software Defined Networking (SDN) is the most expected inventions for the network administrators, cloud service providers and businessmen. In a distant future SDN may bring a revolutionary change in the field of network and research. For its global acceptance, has to undergo security issues and get cover using proper solution .This paper implements the firewall on Spanning Tree Protocol (STP) over Software Defined Networking using Mininet emulator. Implementation of firewall in STP over SDN has been performed by Python based POX Open-Flow controller, OpenvSwitch (OVS) as a forwarding function, Virtual Box, which installed on a Virtual Machine(VM) in Ubuntu Platform. The broadcast streams are controlled by the Firewall on STP based SDN without dedicated hardware support. The result focuses the Firewall functionalities over STP based SDN as well as frees the available bandwidth for traffic.

A ML approach to implement LACP over SDN

Project brief:

Software Defined Networking (SDN) is a directly programmable comprehensive network model that has decoupled the control plane from forwarding plane in the network. Link aggregation (LAG) is the bundling of multiple links into a single aggregated logical link with higher aggregate data bandwidth. This paper presents implementation of Link Aggregation Control Protocol (LACP) over SDN using Mininet Emulator. OpenvSwitch (OVS) acts as a forwarding function, RYU as an OpenFlow controller and Mininet Emulator which is installed on Ubuntu Virtual Machine (VM) in the implementation of LACP in SDN. This paper shows data communication speed improved over SDN using LACP. This paper also addressed that LACP provides inherent automatic redundancy that dynamically redirected to flow across the remaining links while one of the multiple links used in the aggregated groups fail or disabled.
